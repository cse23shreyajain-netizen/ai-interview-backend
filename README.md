# AI Interview Backend

This is the backend of an AI Interview platform.

## 🚀 Features
- User Authentication (Login/Register)
- Interview Management
- Multiple Rounds (Technical, Logical, Behavioral)
- AI-based interview handling

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies:
   npm install

3. Create a .env file and add:
   MONGO_URI=your_mongodb_connection_string
   PORT=5000

4. Run the server:
   npm start

## 🌐 API Base URL
http://localhost:5000

## 📌 Note
Make sure MongoDB is connected before running the server.
